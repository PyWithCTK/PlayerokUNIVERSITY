PlayerokUniversity ⚡️

PlayerokUniversity — это профессиональное настольное приложение для финансового учета и аналитики продаж на маркетплейсе Playerok. Забудьте о калькуляторах: программа сама посчитает профит, комиссии и маржу.

🚀 Как начать работу?

Вариант 1: Готовый файл (.exe) — Рекомендуется

Если вы скачали скомпилированную версию приложения (PlayerokUniversity.exe):

Ничего докачивать не нужно. Никаких библиотек, настроек или баз данных.

Просто перенесите файл в отдельную папку и запустите его.

Важно: При первом сохранении товара программа автоматически создаст файл data.json в той же папке, где находится сам .exe файл. Это ваша база данных, не удаляйте её, если хотите сохранить историю продаж.

Вариант 2: Исходный код (.py)

Если вы запускаете проект через Python (например, в PyCharm):

Установите библиотеку интерфейса:

pip install customtkinter



Запустите скрипт:

python playerok_university.py



Примечание: База данных data.json будет создана автоматически в той же директории, что и ваш скрипт .py.

✨ Основные фишки

Умный расчет прибыли: Автоматическое вычисление суммы «на карту» и чистого профита с учетом всех затрат.

Выбор комиссий в один клик: * Площадка: 10% или 20%.

Вывод: 6% или 10%.

Автономность: Программа сама создаёт и обновляет файл хранения данных (data.json) в корневой папке приложения. Вам не нужно настраивать базы данных вручную.

Современный UI: Чистый бело-синий дизайн, адаптированный под удобную работу с таблицами.

📈 Руководство пользователя

Добавление товара: Введите название, цену на сайте, закуп и стоимость премиума.

Настройка процентов: Нажмите на нужные кнопки комиссий (10/20 или 6/10).

Сохранение: Нажмите кнопку «ДОБАВИТЬ». Данные сразу запишутся в память.

Удаление: Выделите товар в таблице и нажмите красную кнопку «УДАЛИТЬ ВЫБРАННЫЙ ТОВАР» под списком.

Данное ПО создано для автоматизации финансовых расчетов и не является официальным приложением площадки Playerok.
